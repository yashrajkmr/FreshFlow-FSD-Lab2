# FreshFlow | Dynamic Expiry-Driven Pricing Engine

A responsive frontend interface engineered using **HTML5 semantic structures** and **Tailwind CSS** for an e-commerce smart logistics engine. This project addresses real-world food waste by simulating dynamic price markdowns on perishables nearing expiration.

Developed for **FSD Lab: Lab Exercise 2**, Christ (Deemed to be University).

## 🚀 Integrated HTML5 Browser APIs

This repository demonstrates advanced client-side interactivity using **7 native web browser APIs**:

1. **Local Storage API:** Manages persistent user authentication state directly across page reloads.
2. **Geolocation API:** Proximity calculation to find the closest partner supermarket in real-time.
3. **Web Notifications API:** Native desktop system push notifications for rolling flash clearance markdown events.
4. **Clipboard API:** Low-friction secure developer copy utilities for enterprise integration keys.
5. **HTML5 Canvas API:** Custom data visualization tracking real-time programmatic price decay paths.
6. **Drag & Drop API:** Interactive file parsing system allowing store managers to drop custom manifest lists.
7. **HTML5 Video API:** Native media elements for rendering platform diagnostic telemetry demonstrations.
